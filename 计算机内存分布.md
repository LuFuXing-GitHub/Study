计算机内存分布（以32位为例）：
=============================================
地址范围(十六进制)        分区名称          存放内容 & 说明
=============================================
0xC0000000 ~ 0xFFFFFFFF  【内核空间】1GB
                         操作系统内核、驱动、页表
                         ⚠️ 用户态代码**无权直接访问**，仅系统调用进入

---------------------------------------------
0x7FFF0000 ~ 0xBFFFFFFF  【栈 Stack】(线程私有)
                         ✅ 局部变量、函数参数、返回地址、函数栈帧
                         ✅ 向下增长（地址越来越小）
                         ✅ 自动分配/释放，速度快
                         ❗ 默认容量很小，递归过深/大数组会栈溢出

---------------------------------------------
中间大片区域             【共享库 / mmap 映射区】
                         动态链接库(.dll/.so)、内存映射文件、大页内存

---------------------------------------------
0x00400000 ~ 0x7FFF0000  【堆 Heap】(进程共享)
                         ✅ new / malloc 动态申请的内存
                         ✅ 向上增长（地址越来越大）
                         ✅ 手动管理内存，不释放造成内存泄漏
                         ✅ 空间巨大

---------------------------------------------
0x00300000 ~ 0x003FFFFF  【BSS 段】
                         未初始化全局变量、未初始化静态局部变量
                         加载时系统自动清零，磁盘不占空间

---------------------------------------------
0x00200000 ~ 0x002FFFFF  【数据段 Data】
                         已初始化全局变量、已初始化静态局部变量
                         可读可写，程序运行全程存在

---------------------------------------------
0x00000000 ~ 0x001FFFFF  【代码段 Text / 只读段】
                         程序二进制指令、字符串常量、const 只读全局变量
                         权限：只读+可执行，禁止修改，多进程可共享
=============================================

C/C++代码变量归属示意：
// 1. 代码段 Text
const char* str = "hello world";   // 字符串常量 → 代码段
const int g_const = 100;           // 只读全局const → 代码段

// 2. 数据段 Data（已初始化全局/静态）
int g_data = 10;                   // 已初始化全局变量 → 数据段
static int s_data = 20;            // 已初始化静态变量 → 数据段

// 3. BSS段（未初始化全局/静态）
int g_bss;                         // 未初始化全局变量 → BSS
static int s_bss;                  // 未初始化静态变量 → BSS

void test()
{
    // 4. 栈 Stack
    int a = 1;                     // 局部变量 → 栈
    int b[1024];                   // 局部数组 → 栈
    func(a);                       // 函数参数、返回地址 → 栈帧

    // 5. 堆 Heap
    int* p = new int[1000];        // new 出来的内存 → 堆
    char* buf = (char*)malloc(100);// malloc 出来的内存 → 堆

    static int s_local = 30;       // 函数内静态变量(已初始化) → 数据段
    static int s_local2;           // 函数内静态变量(未初始化) → BSS
}

char* p = "test";  // "test" 在 代码段(只读)
p[0] = 'a';        // 运行崩溃！修改只读内存
